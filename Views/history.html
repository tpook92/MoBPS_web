The MoBPS Web-interface is still in an open beta stage!

Version history (last updated 22/10/21):

Our web-application is currently running using MoBPS v1.06.62.
New tutorial videos for YouTube are planned, but currently not the priority!
Updates for the R-package are listed at Github

October 2021 updates:

New Output function: PCA
New option to use PCG Solver for large breeding value estimation

July 2021 updates:

Added additional QTL types for trait generation
Added Tilapia as new potential species

April 2021 updates:

New LD-build up module to simulate founder genotypes
Major database update. Projects can now be arranged in folders and more comfort changes

March 2021 updates:

New input structure more Major QTL position
Added documentation to all advanced modules ("Press here to get info on how this module works!)

Februrar 2021 updates:

OGC now used optiSel (Wellman et al 2019)
Added module to simultaneously edit multiple nodes/edges

November 2020 updates:

Sharing of projects between users
Improved selected screen when using Manuel Select (cohorts for Breeding value estimation)
Added module to control litter size

September 2020 updates:

Style Changes (mostly login-page)
Added display + alert for R warnings
Added feature to copy input from existing node/edges

July 2020 updates:

Added new trait generation options (maternal / paternal / linear combination)
Preprint on the interface at bioRvix
Preprint on the node/edge language to describe breeding programs at bioRvix
Published a MoBPS study on health traits in Horse in Animals
Added maternal / paternal traits

June 2020 updates:

Style Changes (mostly login-page)
Added confidence intervals in Output-modules
Added repeatability as an advanced trait-option

May 2020 updates:

Increased font-size in the flash-application.
Advanced Node-options are now split into sub-options

April 2020 updates:

Release of the CompareProject-Module
MoBPSweb is now a multi-page application

March 2020 updates:

Browser dependent code - MoBPS now supports Safari
The website is still developed and optimized for Google Chrome

February 2020 updates:

Minor style changes

January 2020 updates:

Drag & Drop features for Phenotype information - module
Advanced Phenotypes (offspring avg. etc instead of own phenotypes) for selection
Added option to rescaling of genomic values after burn-in generation (Advanced trait rescaling)
Added Threshold-selection

Release 09.12.2019 (v0.1.1):

Supported R-package = MoBPS v1.4.30
New Web-design for login-page
MoBPS simulation can run without use of miraculix
Added Module to export the population list + population data (vcf/pedmap/phenotypes etc.)
Minor bug-fixes for parallel computing
