Version history:

Our web-application is currently running using MoBPS v1.05.15. We are currently in the process of preparing a new release and a page for our web-application (MoBPSweb). Current target is to submit / release on biovrix in July/August 2020. New tutorial videos for YouTube are planned, but currently not the priority!
