General Information

Project Name ⓘ Enter the name of your project. You can load save/download/delete your project via the action bar and load different version to return to.
Advanced settings
Test-Mode ⓘ Activating this will unlock the scaling of node sizes, allow to run a simulation multiple times and ignore imported genotypes.
Culling Module ⓘ Activating this will unlock the culling module to simulate the death of individuals - that then can not be used for breeding.
Subpopulation Module ⓘ Activating this will unlock the specification of subpopulation with different allele frequencies and trait architectures.
Economic Module ⓘ Activating this will unlock the use of economic parameters.
Advanced Edge/Node options ⓘ Activating this will unlock additional options for nodes/edges (share genotyped, selection options etc.)
Species ⓘ Select the species. This will unlock the correct Ensembl Datasets etc. Use a template to start with already entered somewhat realistic values.
Time Unit ⓘ Select the time unit used (this is only relevant for economic calculation). Time of generation for each cohort will be displayed.
Test/Speed Mode ⓘ Activate this to set the number of repeats to 1, reduce the genome length to 10M and 200 SNPs and deactivate data import. This should heavily reduce computation times for testing
Individual numbers scaling ⓘ Scale the number of individuals in each node by this factor. This can further reduce comp. times! Make sure each node contains at least 1 individual ( do not worry about point numbers - round() is used)
Multi-Simulation Mode ⓘ Activate this to run the simulation multiple times. This will heavily increase running time but also reduce variance of the results.
Number of Simulations ⓘ Number of simulations of the breeding program you what to perform.
Number of Simulations run in parallel ⓘ Number of simulations of the breeding program to run simultaneously.
Genetic Data ⓘ Select the genome you want to work with.

Ensembl Dataset ⓘ Currently only the genomic maps available in MoBPSmaps are provided. To add an additional map here contact Torsten or upload your own. Please choose a species first.
Max. Number of SNPs ⓘ Only a subsample of the Ensembl dataset is used to save computation time. If nothing is enter use the full dataset.
Upload Map File ⓘ Currently only .vcf .map and .RData-files are supported. Adding more is no problem. Contact Torsten if required.
